## Estrutura

```
├── index.html              → o site inteiro (HTML + CSS + JS no mesmo arquivo)
└── assets/
    ├── logo-circular.png   → logo da marca
    ├── hero-mural.png      → arte usada na seção "Sobre"
    └── produtos/           → fotos dos produtos (adicione aqui)
```

## Como editar produtos

Abra o `index.html` em qualquer editor de texto e procure por `PAINEL DE PRODUTOS`
(use Ctrl+F). Todas as categorias e produtos em destaque ficam ali, num formato
simples de copiar/colar — não precisa saber programar. Instruções completas estão
comentadas dentro do próprio arquivo.

## Como publicar gratuitamente com o GitHub Pages

1. Suba este repositório para o GitHub (se ainda não subiu)
2. Vá em **Settings → Pages**
3. Em **Source**, selecione a branch `main` e a pasta `/root`
4. Salve — em alguns minutos o site fica no ar em `https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO/`
